Tutorial de Uso – Cuadratura Gauss-Legendre

En este tutorial se muestra cómo utilizar el módulo "gauss_quadrature.py" para aproximar integrales mediante la cuadratura Gauss-Legendre, incluyendo la aplicación de expansión de Taylor para funciones trascendentales.

Aproximación de una integral definida

Consideremos la función

\[ \phi(t) = t^6 - t^2 \sin(2t) \]

y el intervalo de integración [1,3]. Podemos aproximar su integral usando N nodos de Gauss-Legendre

Posible implementación en Python

from gauss_quadrature import funcion, obtener_puntos_pesos_reescalados

import numpy as np

N = 4 # Número de nodos de Gauss-Legendre

a, b = 1, 3 # Límites de integración

puntos, pesos = obtener_puntos_pesos_reescalados(a, b, N)

Evaluar la integral numéricamente, se escribiría como

resultado = np.sum(pesos*funcion(puntos))

print(f"Integral aproximada con N={N} es {resultado:.8f}")

Teniendo una integral aproximada con \(N=4, \ 578.66666667\) siendo un valor esperable.

Así, obtener_puntos_pesos_reescalados(a, b, N) devuelve los puntos óptimos y sus pesos correspondientes en el intervalo [a,b]. De forma tal que, la integral se aproxima como la suma ponderada \(\sum_{i=1}^N w_i \phi(x_i)\).

El número de nodos \(N\) afecta directamente la precisión del resultado.

from gauss_quadrature import funcion, obtener_puntos_pesos_reescalados import numpy as np

a, b = 1, 3

print("Comparación de precisión con diferentes valores de N:") print("N\tResultado\t\tDiferencia") print("-" * 50)

Se calcula la solución de referencia, cuya cual, es la analítica.

from gauss_quadrature import solucion_analitica ref = float(solucion_analitica)

for N in range(2, 7): puntos, pesos = obtener_puntos_pesos_reescalados(a, b, N) resultado = np.sum(pesos * funcion(puntos)) error = abs(resultado - ref) print(f"{N}\t{resultado:.6f}\t\t{error:.2e}")

print(f"\nValor de referencia (analítico): {ref:.8f}")

El valor de referencia analítico es aproximadamente \(578.66666667\). Con \(N \leqslant 3\) se alcanza precisión numérica máxima para esta función.

Para funciones que incluyen términos trascendentales, podemos usar expansiones de Taylor.

from gauss_quadrature import funcion_taylor, obtener_puntos_pesos_reescalados import numpy as np

a, b = 1, 3 N = 4 grado_taylor = 4 # Grado de la expansión de Taylor

puntos, pesos = obtener_puntos_pesos_reescalados(a, b, N)

Se evalúa usando expansión de Taylor

valores, expr_polinomio = funcion_taylor(puntos, grado_taylor)

resultado_taylor = np.sum(pesos * valores)

print(f"Aproximación con el teorema de Taylor (grado {grado_taylor}) {resultado_taylor:.8f}")

print(f"La expresión polinómica equivalente es {expr_polinomio}")

Sin embargo, para un uso más sencillo, podemos emplear la función calcular_integral_gaussiana.

from gauss_quadrature import funcion, calcular_integral_gaussiana

resultado = calcular_integral_gaussiana(funcion, 1, 3, 5)

print(f"Resultado con función de conveniencia dada {resultado:.8f}")

Probando con diferentes valores de N

print("\nPrueba rápida con diferentes N")

for N in [2, 3, 4, 5]:

resultado = calcular_integral_gaussiana(funcion, 1, 3, N)

print(f"N={N}: {resultado:.8f}")

Si se buscara con mayor precisión podría ser

from gauss_quadrature import funcion, calcular_integral_gaussiana, solucion_analitica import numpy as np

def encontrar_N_optimo(tolerancia=1e-10): """Encuentra el N mínimo que satisface una tolerancia de error""" ref = float(solucion_analitica)

for N in range(2, 10):
    resultado = calcular_integral_gaussiana(funcion, 1, 3, N)
    error = abs(resultado - ref)

    if error <= tolerancia:
        print(f"N={N} satisface la tolerancia dada error = {error:.2e}")
        return N
    else:
        print(f"  N={N} con error = {error:.2e}")

print("No se alcanzó la tolerancia con N <= 9")
return None

Se ejecuta para hallar un posible N que satifaga las condiciones.

print("Búsqueda del N óptimo (tolerancia = 1e-10):") encontrar_N_optimo()

La búsqueda del \(N\) óptimo con tolerancia igual a \(10^{-10}\)

\[ \begin{array}{ll} N=2: & \text{error } = 6.22 \times 10^{0} \\ N=3: & \text{satisface la tolerancia } (1.78 \times 10^{-14}) \end{array} \]

Así, la cuadratura de Gauss con \(N\) puntos integra exactamente polinomios de grado \(2N-1\).

N Grado máximo exacto
2 3
3 5
4 7
5 9
6 11

\(\phi(t) = t^6 - t^2 \sin(2t)\) tiene una parte polinomial de grado 6. Para integrar exactamente un polinomio de grado 6, necesitamos al menos N=4 (que integra exactamente hasta grado 7).
